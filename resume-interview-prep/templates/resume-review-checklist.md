# Resume Review Checklist

Use this checklist to review a resume before submission. Copy this file and check off items as you go.

## Formatting
- [ ] Clean, consistent formatting throughout
- [ ] Standard section headers (Experience, Education, Skills)
- [ ] No tables, text boxes, or graphics (ATS-safe)
- [ ] Appropriate length (1 page for <10 years, 2 pages max for senior)
- [ ] Consistent date formatting
- [ ] Readable font (10-12pt body)
- [ ] Sufficient white space
- [ ] Contact info includes name, city/state, phone, email, LinkedIn

## Content — Bullet Points
- [ ] Every bullet starts with a strong action verb
- [ ] Bullets describe accomplishments, not duties
- [ ] Results are quantified (numbers, percentages, dollar amounts)
- [ ] Most impactful bullets listed first under each role
- [ ] 3-5 bullets per role (more for recent, fewer for older)
- [ ] No first-person pronouns (I, me, my)

## Tailoring
- [ ] Keywords from the job posting are incorporated naturally
- [ ] Professional summary (if used) addresses the target role
- [ ] Skills section matches the posting's requirements
- [ ] Most relevant experience is emphasized
- [ ] Irrelevant experience is trimmed or removed

## ATS Compatibility
- [ ] Standard section headers used
- [ ] Keywords appear in context (not just skills list)
- [ ] Both acronyms and full terms included (e.g., "SQL (Structured Query Language)")
- [ ] File saved as .docx or .pdf per application requirements
- [ ] Text is parseable in plain text view

## Proofreading
- [ ] No spelling errors
- [ ] No grammatical errors
- [ ] Consistent tense (past for prior roles, present for current)
- [ ] No jargon that the reader would not understand
- [ ] Phone number and email are correct
- [ ] LinkedIn URL works
