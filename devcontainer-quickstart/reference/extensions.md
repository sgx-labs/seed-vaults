# Recommended VS Code Extensions for Devcontainers

| Extension | ID | Purpose |
|-----------|-----|---------|
| Claude Code | `anthropic.claude-code` | AI coding assistant |
| GitLens | `eamodio.gitlens` | Git blame, history, annotations |
| Git Graph | `mhutchie.git-graph` | Visual git log |
| Error Lens | `usernamehw.errorlens` | Inline error/warning display |
| Todo Tree | `gruntfuggly.todo-tree` | Track TODO/FIXME/SECURITY/PII tags |
| Material Icons | `PKief.material-icon-theme` | File icons |
| EditorConfig | `EditorConfig.EditorConfig` | Consistent formatting |
