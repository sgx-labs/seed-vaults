# Decision Log

## AD-001: Use hub-first navigation pattern

**Status:** Accepted
**Date:** 2025-03-06

Organize seed content using a hub-first cascade: hub notes provide overviews, research notes go deep on specific questions, entity notes serve as quick references. This mirrors how people naturally learn -- start broad, then drill down.

**Alternative considered:** Flat structure with all notes in one directory. Rejected because it makes discovery harder and doesn't guide the user through topics in a logical order.
