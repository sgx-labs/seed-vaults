# SAME Seed Vaults

Pre-built knowledge vault templates for SAME (Stateless Agent Memory Engine). Each seed vault is a curated collection of markdown notes on a specific topic.

## Structure

Each top-level directory is a standalone seed vault containing:
- A `README.md` describing the vault's contents
- Markdown notes organized by topic

`seeds.json` is the manifest listing all available seed vaults with metadata.

## No Build Step

This is a content-only repository. All files are markdown.

## Contributing

1. Create a new top-level directory for your seed vault
2. Add a `README.md` with a description and note count
3. Add your markdown notes
4. Update `seeds.json` with the new entry
5. Open a pull request against `main`
